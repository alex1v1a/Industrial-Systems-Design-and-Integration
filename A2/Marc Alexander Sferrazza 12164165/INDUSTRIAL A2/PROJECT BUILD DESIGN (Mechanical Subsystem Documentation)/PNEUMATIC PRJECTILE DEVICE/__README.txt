Included in this documentation is all the required files to build a pneumatic projectile device.

Please see the "Pneumatic Projectile Device" report in this directory for a full overview of this project and the reasoning behind the build choices.

The containing folders for all required components can be found in "BUILD" for all necessary physical parts, with "PCB" for all electronic parts, and "PROGRAM" with the required software for the device, ready to edit as per user preference.

Please see the README files in each folder for further explanation.
